KOReader Sync Setup

Configure progress sync on any device running KOReader (Kindle, Mac, etc.):

  1. Open any book → tap top of screen → Tools → Progress sync
  2. Set Server to http://192.168.1.10:7200
  3. Enter a Username and Password
  4. Tap Register (first time only), then Login

Use the same credentials on all devices. Local network or VPN required.
